HOW-TO INSTALL:
(Automatic execution)
•	Remotely upload the ZIP file via Administration interface, or place the ZIP file to MARS root folder
•	Wait for next Scheduler execution (each 15 minutes)
•	Check schedule.log/update-*.log for results
(Manual execution)
•	Extract the ZIP file to MARS folder with overwrite
•	Open command prompt in MARS\#*
•	Run .CMD
•	Delete the #* folder once finished.
